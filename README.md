# 好友派对助手

## 项目简介

**好友派对助手** 是一款基于 uni-app 开发的微信小程序应用，旨在为好友聚会提供便捷的游戏和互动工具。该应用集成了多种桌游辅助功能、派对小游戏、好友排名系统以及匿名社交功能，让每一次聚会都更加精彩有趣。

## 功能模块

### 1. 桌游助手

#### 1.1 狼人杀房间
- **房间管理**：创建房间、加入房间、邀请好友
- **角色配置**：预设板子选择、自定义角色配置
- **胜利条件**：标准屠边、屠城模式
- **发牌系统**：随机发牌、身份查看
- **裁判模式**：击杀/投票出局、复活玩家
- **游戏结算**：自动判定胜负、显示玩家战绩
- **网络功能**：实时数据同步、房间邀请、网络状态指示

#### 1.2 阿瓦隆房间
- **角色系统**：梅林、派西维尔、忠臣、刺客、莫甘娜、莫德雷德、奥伯伦、爪牙
- **人数配置**：支持 5-10 人游戏
- **游戏流程**：组队 → 投票 → 任务执行 → 刺杀阶段
- **特殊机制**：第五人规则、刺杀梅林翻盘
- **信息可见**：梅林看坏人、派西维尔看梅林/莫甘娜、坏人互认

### 2. 派对工具箱

#### 2.1 掷骰子
- 支持 1-6 个骰子
- 动画效果展示
- 随机结果生成

#### 2.2 计时器
- 正计时/倒计时模式
- 自定义时间设置
- 提醒功能

#### 2.3 真心话大冒险
- 真心话问题库
- 大冒险挑战库
- 随机抽取功能

### 3. 好友排名

- **房间系统**：创建排名房间、邀请好友加入
- **积分记录**：记录每局游戏得分
- **排行榜**：实时更新好友排名
- **数据持久化**：本地存储历史数据

### 4. 匿名吐槽

- **匿名发言**：随机生成匿名昵称和头像
- **互动功能**：点赞、点踩、回复
- **聊天室 UI**：类似聊天室的消息展示
- **本地存储**：消息持久化保存

## 使用说明

### 1. 安装运行
1. 下载项目代码
2. 使用 HBuilderX 打开项目
3. 运行到微信开发者工具或手机模拟器

### 2. 打包发布
1. 配置 manifest.json 文件
2. 设置应用图标
3. 使用 HBuilderX 云打包
4. 生成 APK 或小程序代码

### 3. 功能使用
- **桌游助手**：选择游戏类型 → 创建/加入房间 → 开始游戏
- **派对工具箱**：选择工具 → 开始使用
- **好友排名**：创建房间 → 邀请好友 → 记录得分
- **匿名吐槽**：直接发言 → 互动交流
